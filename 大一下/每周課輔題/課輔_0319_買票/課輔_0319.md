# 📝 課輔實作 (03/19) - 繼承與票價系統：買票邏輯

## 💡 實作重點與解題思路

本次課輔題目要求建立一個票務系統，根據票種（全票/學生票）與會員身分來決定最終票價。

### 1. 受保護成員的繼承 (Protected Inheritance)
* **父類別 `ticket`**：定義了保護屬性 (`protected`) `type` (票種名稱) 與 `isMember` (是否為會員)。
* **權限特性**：使用 `protected` 能確保這些變數在子類別 `order` 中可以被直接操作，但在類別外部則受到保護，符合封裝精神。

### 2. 條件判斷與優惠規則
* **票種定價**：全票(A)為 300 元，學生票(S)為 200 元。
* **優惠邏輯**：
    * 若為會員且購買全票，享 **8 折** 優惠 (300 * 0.8 = 240)。
    * 若為會員且購買學生票，享 **9 折** 優惠 (200 * 0.9 = 180)。
    * 非會員則維持原價輸出。

### 3. 實作細節
* 程式同時支援了大、小寫字母的輸入判定（如 'a' 與 'A'），提升了程式的容錯率與穩定性。

---

## 💻 完整程式碼實作與註解牆

```cpp
#include <iostream>
using namespace std;

class ticket {
protected:
    char type;                                              // 受保護屬性：票種 (A/S)
    char isMember;                                          // 受保護屬性：會員狀態 (Y/N)
};

class order : protected ticket {
public:
    int price;                                              // 公有屬性：最終票價

    // 建構子：根據傳入參數初始化並計算票價
    order(char a, char b) {
        type = a;                                           // 初始化票種
        isMember = b;                                       // 初始化會員狀態
        
        if (b == 'Y' || b == 'y') {                         // 判定為會員的情況
            if (a == 'a' || a == 'A') {
                price = 240;                                // 全票原價300，會員打8折 = 240
            }
            else {
                price = 180;                                // 學生票原價200，會員打9折 = 180
            }
        }
        else {                                              // 非會員的情況，維持原價
            if (a == 'a' || a == 'A') {
                price = 300;                                // 全票原價輸出
            }
            else {
                price = 200;                                // 學生票原價輸出
            }
        }
    }

    // 顯示購買結果
    void display() {
        cout << "票種:" << type << endl;
        cout << "會員狀態:" << isMember << endl;
        cout << "最終金額:" << price << "元" << endl;
    }
};

int main() {
    char a;                                                 // 宣告變數儲存票種輸入
    char b;                                                 // 宣告變數儲存會員身分輸入
    
    cin >> a >> b;                                          // 讀取使用者輸入
    order p1(a, b);                                         // 實體化訂單物件並執行運算
    p1.display();                                           // 呼叫顯示方法
    
    return 0;                                               // 程式結束
}
