# 🤝 課輔實作 (04/02) - 運算子多載：長度相加系統 (Length Addition)

## 💡 實作重點與解題思路

本次課輔實作要求建立一個長度類別 `Length`，透過 **運算子多載** 實作兩個長度（包含公尺與公分）的相加，並處理進位與格式化輸出。

### 1. 進位邏輯處理( 非最優解! )
* **單位換算**：定義 1 公尺 (M) 等於 100 公分 (cm)。
* **進位觸發**：在執行 `operator+` 運算時，若相加後的公分部分 `v.cm >= 100`，則公尺部分 `v.m` 加 1，並將公分部分減去 100。

### 2. 運算子多載 (`operator+`)
* **物件封裝**：將公尺與公分的累加邏輯封裝在類別內，讓 `main` 函式可以使用 `L3 = L1 + L2` 這種簡潔的語法。
* **暫存物件**：同樣利用空建構子建立一個臨時物件 `v` 來存放運算結果並回傳。

### 3. 輸出格式判定
* 根據題目要求的輸出規範，`show()` 方法會判斷公分是否為 0：
    1. **整數公尺**：若公分為 0，僅輸出 `m M`。
    2. **一般格式**：若公分非 0，輸出 `m M cm cm`。

---

## 💻 完整程式碼實作與註解牆

```cpp
#include<iostream>
using namespace std;

class Length {
private:
    int m;                                                  // 私有屬性：公尺 (Meter)
    int cm;                                                 // 私有屬性：公分 (Centimeter)

public:
    // 預設建構子：提供運算子多載內部建立結果物件使用
    Length() {                                              // 保留空白的空建構子

    }

    // 參數建構子：初始化公尺與公分數值
    Length(int a, int b) {
        m = a;
        cm = b;
    }

    // 【運算子多載】：定義兩個 Length 物件相加 (+) 的行為
    Length operator+(Length n) {
        Length v;                                           // 建立暫存結果物件
        v.m = m + n.m;                                      // 公尺部分直接相加
        v.cm = cm + n.cm;                                   // 公分部分直接相加
        
        if (v.cm >= 100) {                                  // 進位判定：若公分滿 100
            v.m++;                                          // 公尺進位 1
            v.cm = v.cm - 100;                              // 修正公分數值
        }
        return v;                                           // 回傳計算後的長度物件
    }

    // 格式化顯示結果
    void show() {
        if (cm == 0) {                                      // 情況 1：剛好整除或原本公分為 0
            cout << m << "M " << endl;                      // 僅輸出公尺部分
        }
        else {                                              // 情況 2：包含公分剩餘值
            cout << m << "M " << cm << "cm" << endl;        // 完整輸出 M 與 cm
        }
    }
};

int main() {
    int m1, cm1, m2, cm2;
    
    // 讀取兩組長度輸入 (M1 cm1 M2 cm2)
    if (cin >> m1 >> cm1 >> m2 >> cm2) {
        Length L1(m1, cm1);                                 // 實體化長度 1
        Length L2(m2, cm2);                                 // 實體化長度 2
        
        Length L3 = L1 + L2;                                // 執行物件加法 (觸發 operator+)
        
        L3.show();                                          // 顯示累加後的長度
    }
    
    return 0;                                               // 程式結束
}
