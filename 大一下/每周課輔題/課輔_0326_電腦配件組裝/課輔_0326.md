# 🤝 課輔實作 (03/26) - 多重繼承：電競電腦組裝系統 (PC Builder)

## 💡 實作重點與解題思路

本次課輔題目模擬電腦組裝過程，要求透過繼承兩個父類別來計算硬體總價。

### 1. 資料結構與封裝
* **`Part` 結構體**：定義了硬體型號 (`Model`) 與價格 (`cost`) 的對應關係。
* **元件類別**：分別建立 `CPU` 與 `GPU` 類別，內部維護各自的零件陣列與搜尋函式 `find_price`。

### 2. 多重繼承架構
* **子類別實作**：`class myPC : public CPU, public GPU`。
* **邏輯整合**：`myPC` 的建構元接收兩個字串參數，分別呼叫父類別的 `find_price` 函式進行價格檢索並累加。

### 3. 搜尋演算法
* 採用線性搜尋法遍歷陣列。
* 比對輸入的型號名稱，若匹配成功則回傳該零件價格，若無匹配則回傳 0。

---

## 💻 完整程式碼實作

```cpp
#include<iostream>
using namespace std;

struct Part {                                               // 定義零件結構體
    string Model;                                           // 零件型號
    int cost;                                               // 零件價格
};

class CPU {                                                 // CPU 基底類別
private:
    Part p[3] = { {"i5",100},{"i7",200},{"i9",300} };       // CPU 零件清單
    int out;                                                // 內部狀態變數
public:
    int find_price(string);                                 // 搜尋 CPU 價格介面
};

class GPU {                                                 // GPU 基底類別
private:
    Part g[3] = { {"4060",200},{"4070",400},{"4080",600} }; // GPU 零件清單
    int out;                                                // 內部狀態變數
public:
    int find_price(string);                                 // 搜尋 GPU 價格介面
};

// --- 成員函式實作 ---

int CPU::find_price(string a) {                             // 實作 CPU 價格檢索
    for (int i = 0; i < 3; i++) {                           // 遍歷清單
        if (a == p[i].Model) {                              // 型號比對成功
            return p[i].cost;                               // 回傳對應價格
        }
    }
    return 0;                                               // 未找到則回傳 0
}

int GPU::find_price(string b) {                             // 實作 GPU 價格檢索
    for (int i = 0; i < 3; i++) {                           // 遍歷清單
        if (b == g[i].Model) {                              // 型號比對成功
            return g[i].cost;                               // 回傳對應價格
        }
    }
    return 0;                                               // 未找到則回傳 0
}

// --- 多重繼承衍生類別 ---

class myPC : public CPU, public GPU {                       // 同時繼承 CPU 與 GPU 類別
private:
    int cost;                                               // 儲存組裝總成本
public:
    myPC(string a, string b) {                              // 建構子：執行總價計算
        cost = CPU::find_price(a) + GPU::find_price(b);     // 累加父類別回傳的價格
    }
    void show() {                                           // 顯示結果方法
        cout << "Total Cost:" << cost;                      // 輸出格式化結果
    }
};

int main() {
    string cpu_name, gpu_name;                              // 宣告輸入字串
    cin >> cpu_name >> gpu_name;                            // 讀取型號輸入
    myPC pc1(cpu_name, gpu_name);                           // 建立電腦物件並計算價格
    pc1.show();                                             // 顯示最終總成本
    return 0;                                               // 程式結束
}
