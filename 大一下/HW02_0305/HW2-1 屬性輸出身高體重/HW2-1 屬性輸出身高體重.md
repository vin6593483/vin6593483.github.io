# 作業 2-1：類別基礎與屬性封裝 (Class & Encapsulation)

本練習正式踏入 C++ 物件導向程式設計 (OOP) 的領域。程式透過定義 `person` 類別，將資料**屬性**（身高、體重）與**行為**函數（設定資料、顯示資訊）整合在一起。

## 🔍 核心觀念解析

### 1. 存取修飾詞：`private` 與 `public`
* **`private` (私有)**：將 `high` 與 `weight` 設為私有，防止外部程式碼直接修改內部資料。
* **`public` (公開)**：提供外部呼叫的介面（ `setup` 和 `show`），確保資料只能透過預設的方式被讀取或更動。

### 2. 資料驗證 (Input Validation)
在 `show()` 函數中，程式對身高與體重進行了邏輯檢查。若數值為負數（不合常理），則輸出錯誤。

---

## 💡 完整程式碼(註解)

```cpp
#include<iostream>
using namespace std;

class person {                                        // 定義一個名為 person 的類別
private:                                              // 私有區段：外部無法直接存取
  int high;                                         // 儲存身高屬性
	int weight;                                       // 儲存體重屬性
public:                                               // 公開區段：外面可以調用的函數
	void setup(int m, int n) {                        // setup函數
		high = m;                                     // 將參數 m 存入私有屬性 high
		weight = n;                                   // 將參數 n 存入私有屬性 weight
	}
	void show() {                                     // show函數
		if (high < 0 || weight < 0) {                 
			cout << "他不是人" << endl;                 // 輸出錯誤
        else {                                        // 若是資料正常
			cout << "身高" << high << endl;            // 印出身高資訊
			cout << "體重" << weight << endl;          // 印出體重資訊
		}
	}
};

int main() {                                          // 進入主程式
	person p[2];                                      // 建立一個包含 2 個 person 物件的陣列
	int h1, w1, h2, w2;
	cin >> h1 >> w1 >> h2 >> w2;                      // 讀取使用者輸入的兩組數據
	p[0].setup(h1, w1);                               // setup第一位物件的屬性
	p[1].setup(h2, w2);                               // setup第二位物件的屬性
	for (int i = 0; i < 2; i++) {                     // 透過迴圈走訪物件陣列
		cout << "第" << i + 1 << "位:\n";
        p[i].show();                                  // 呼叫show函數輸出結果
	}
	return 0;                                         
}
