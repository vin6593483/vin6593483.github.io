# 作業 2-3 (加分題)：找字機 (Word Finder)

本程式實作了一個基礎的文本解析類別 `find_word`。使用者輸入一個特定字元後，程式會自動掃描預設的句子，並找出所有「包含該字元」的單字，將其逐一印出並統計總數。

## 🔍 核心觀念解析

### 1. 字串切分邏輯
程式利用迴圈走訪字元陣列，並以空格 `' '` 與句點 `'.'` 作為分隔符號。
* **`start` 變數**：紀錄每個單字在陣列中的起始索引。
* **邊界判斷**：當偵測到分隔符號時，程式會鎖定 `[start, i)` 這個區間作為一個獨立的單字進行處理。

### 2. 巢狀迴圈與搜尋
* **內層搜尋**：一旦鎖定單字區間，立即啟動第二層迴圈搜尋目標字元 `x`。
* **即時輸出與中斷**：若找到目標字元，則使用第三層迴圈印出整個單字，並使用 `break` 跳出搜尋，避免同一個單字因為包含多個目標字元而被重複統計。

---

## 💡 完整版程式碼(含註解)

```cpp
#include<iostream>
using namespace std;

class find_word {                                     // 定義找字類別
public:
	int find(char x, char* a, int length) {           // 核心成員函數：搜尋字元並印出單字
		int start = 0;                                // 紀錄單字起始位置
		int count = 0;                                // 統計包含目標字元的單字總數
		for (int i = 0; i <= length; i++) {           // 走訪整段字串
			if (a[i] == ' ' || a[i] =='.') {          // 偵測到空格或句點（單字結束）
				for (int j = start; j < i; j++) {     // 針對當前切分出的單字區間進行掃描
					if (a[j] == x) {                  // 若單字中包含目標字元 x
						 count++;                     // 統計數加 1
						 for (int k = start; k < i; k++) { // 啟動印出迴圈
							cout << a[k];             // 逐字印出該單字
						 }
						 cout << endl;                // 換行
						 break;                       // 找到即跳出，避免同單字重複統計
					}
				}
				start = i + 1;                        // 更新下一個單字的起始位置
			}
		}
		return count;                                 // 回傳總共找到的單字數
	}
};

int main() {                                         
	char a[] = "I am John. I love apple and water.";  // 預設搜尋文本
	int length = sizeof(a);                           // 取得字串總長度
	char x;                                           // 宣告目標字元變數
	cin >> x;                                         // 讀取使用者要搜尋的字元
	find_word f1;                                     // 調用找字函數
	cout << f1.find(x, a, length) << endl;            // 執行搜尋並印出最終統計數量
	return 0;                                         
}
