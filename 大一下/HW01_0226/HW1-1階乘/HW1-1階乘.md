# 作業 1-1：階乘計算 (Factorial)

這是大一下學期的第一個暖身練習。本程式實作了數學中的階乘運算 (n!)，透過自訂函數 `f(int n)` 與 `for` 迴圈進行迭代累乘。程式不僅計算出最終的數值，還在運算過程中同步輸出完整的數學展開式（例如：`5*4*3*2*1=120`），清楚展示了迴圈執行的軌跡。

## 💡 程式碼與解題思路

詳細的迴圈邏輯與輸出排版，已對齊在右側註解區：

```cpp
#include<iostream>
using namespace std;
int f(int n) {                                // 自訂函數：處理階乘計算與過程輸出
	int sum = 1;                              // 宣告 sum 用於儲存累乘結果，初始值必須為 1
	cout << n;                                // 為了排版美觀，先印出開頭的第一個數字 n
	for (int i = n; i > 1; i--) {             // 迴圈從 n 開始遞減，直到 2 為止，(不到一的原因一個是因為我們輸出的是i-1，另一個是乘一可以省去不乘)
		sum = sum * i;                        // 將當前的數字乘入 sum 中
		cout << "*" << i - 1;                 // 印出乘號與下一個即將相乘的數字
	}
	return sum;                               // 迴圈結束後，回傳最終的階乘總和
}
int main(void) {                              // 進入主程式
	int a;
	cin >> a;                                 // 讀取使用者輸入的整數
	int b = f(a);                             // 呼叫函數 f 進行計算，並將回傳值存入 b
	cout << "=" << b << endl;                 // 印出等號與最終計算結果，完成完整算式
	return 0;                                 // 程式順利結束
}
```
