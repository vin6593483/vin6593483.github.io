# 作業 1-2：學生成績統整系統

這份作業練習了如何使用自訂結構 (`struct`) 來管理多重屬性的資料。程式宣告了一個包含「性別」與「成績」的結構體，並建立大小為 5 的結構陣列來儲存學生資料。透過迴圈讀取輸入後，自動將男女生分流，並計算出各自的平均成績。

## 💡 程式碼與解題思路

將計算人數的變數直接宣告為 `float`，可確保最後平均值計算時保有小數點的精確度。詳細邏輯已對齊在右側註解：

```cpp
#include<iostream>
using namespace std;
struct student {                                // 定義學生結構體
	char gender;                                // 宣告字元變數儲存性別 ('b' 代表男生，其他代表女生)
	float score;                                // 宣告浮點數變數儲存成績
};
int main(void) {                                // 進入主程式
	student s[5];                               // 建立一個大小為 5 的 student 結構陣列
	float boyscore = 0;                         // 儲存男生的總分
	float girlscore = 0;                        // 儲存女生的總分
	float boynum = 0;                           // 儲存男生人數 (宣告為 float 以利後續精確除法)
	float girlnum = 0;                          // 儲存女生人數
	
	for (int i = 0; i < 5; i++) {               // 執行 5 次迴圈讀取全班資料
		cin >> s[i].gender;                     // 讀取第 i 位學生的性別
		cin >> s[i].score;                      // 讀取第 i 位學生的成績
		
		if (s[i].gender == 'b') {               // 判斷若為男生
			boynum++;                           // 男生人數加 1
			boyscore = boyscore + s[i].score;   // 將該生成績加入男生總分
		}
		else {                                  // 若非男生 (視為女生)
			girlnum++;                          // 女生人數加 1
			girlscore = girlscore + s[i].score; // 將該生成績加入女生總分
		}
	}
	
	cout << boyscore / boynum << endl;          // 計算並輸出男生平均分數
	cout << girlscore / girlnum;                // 計算並輸出女生平均分數
	
	return 0;                                   // 程式順利結束
}
```
