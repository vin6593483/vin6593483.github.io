# 🧮 加分題：大數乘法 (Big Number Multiplication)

在 C++ 標準型態中，`int` 或 `long long` 都有其存儲極限。當我們需要運算超過 20 位數的乘法時，必須透過「陣列」模擬人類的直式運算邏輯。本實作的核心在於**空間動態配置**、**反向存儲**與**位數交叉相乘**。

---

## 🔍 核心觀念說明

### 1. 為什麼要「反向放入」陣列？
這是大數運算最精妙的地方。當輸入 `123` 時，字面上 `1` 是最高位（百位），但在程式邏輯中，我們習慣將個位數放在 `arr[0]`。
* **對齊優勢**：不論兩個數字長度是否相同，`arr[0]` 永遠是個位數，這讓運算時的位數對齊變得極其簡單。
* **進位直覺**：當 `arr[0]` 產生進位時，數值會往 `arr[1]`（十位）傳遞，這與陣列索引增加的方向一致，不需頻繁搬移記憶體。

### 2. 交叉相乘與統一進位
在直式乘法中，被乘數的第 $i$ 位與乘數的第 $j$ 位相乘，其結果必然貢獻在積的第 $i+j$ 位。
* **延遲進位**：我們先將所有的乘法結果累加在 `sum[i+j]` 中，此時該位置的值可能會超過 10（例如 45）。
* **規整化**：最後再用一個迴圈統一處理進位，將 `sum[i] / 10` 加到下一位，本位則保留 `sum[i] % 10`。

要想通的地方是當我們將數字反向儲存，陣列索引正好對應權重。當 arr1[i] 與 arr2[j] 相乘時，其結果的權重必然是 $10^{i+j}$，因此我們直接將乘積累加到 sum[i+j] 這格，這就是直式乘法的位移邏輯。

## 💡 完整版程式碼 (有註解版)

```cpp
#include<iostream>
using namespace std;

int main(void) {
	int n = 0;                                          // 宣告被乘數位數變數
	cin >> n;                                          // 讀取位數
	int* arr1 = (int*)malloc(n * sizeof(int));          // 動態配置被乘數空間
	for (int i = 0; i < n; i++) arr1[i] = 0;            // 初始化陣列
	for (int i = n-1; i >= 0; i--) cin >> arr1[i];      // 【關鍵】反向讀取：讓 arr1[0] 存儲個位數

	if (arr1[n-1] == 0) {                               // 安全檢查：最高位不應為 0
		cout << "最高位不可為0!";
		return 0;
	}

	int m = 0;                                          // 宣告乘數位數變數
	cin >> m;                                           // 讀取位數
	int* arr2 = (int*)malloc(m * sizeof(int));          // 動態配置乘數空間
	for (int i = 0; i < m; i++) arr2[i] = 0;            // 初始化陣列
	for (int i = m-1; i >= 0; i--) cin >> arr2[i];      // 【關鍵】反向讀取：讓 arr2[0] 存儲個位數

	if (arr2[m-1] == 0) {                               // 安全檢查：最高位不應為 0
		cout << "最高位不可為0!";
		return 0;
	}

	int* sum = (int*)malloc((m + n) * sizeof(int));     // 積的最大位數為兩數之和
	for (int k = 0; k < (m + n); k++) sum[k] = 0;       // 初始化結果陣列

	for (int i = 0; i < n; i++) {                       // 遍歷被乘數每一位
		for (int j = 0; j < m; j++) {                   // 遍歷乘數每一位
			sum[i + j] += arr1[i] * arr2[j];            // 【核心】交叉相乘並累加至對應位數
		}
	}

	for (int i = 0; i < (m + n - 1); i++) {             // 統一處理進位邏輯
		sum[i + 1] = sum[i + 1] + (sum[i] / 10);        // 將超過 10 的部分進位至下一位
		sum[i] = sum[i] % 10;                           // 本位僅保留個位數
	}

	int check = 0;                                      // 用於排除前導零的標記
	for (int i = (m + n - 1); i >= 0; i--) {            // 從最高位索引開始倒著印出
		if (check == 0 && sum[i] == 0) {                // 若尚未印出有效數字且目前為 0
			check++;                                    // 略過此位 (處理可能的多餘空間)
		}
		else {                                          // 一旦遇到非零數字或已開始印出
			cout << sum[i];                             // 正常輸出數位
		}
	}

	free(arr1);                                         // 釋放動態記憶體 (Good Practice)
	free(arr2);                                         // 釋放動態記憶體
	free(sum);                                          // 釋放動態記憶體
	return 0;                                           // 程式結束
}
