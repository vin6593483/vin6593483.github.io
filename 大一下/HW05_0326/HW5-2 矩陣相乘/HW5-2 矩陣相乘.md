# 🎓 HW5-2：運算子多載 (Operator Overloading) - 2x2 矩陣相乘系統

## 📝 實作重點與解題思路

本次實作的核心在於定義矩陣類別 `matrix`，並透過 **運算子多載** 實作 2x2 矩陣的乘法邏輯。

### 1. 矩陣乘法邏輯
* **定義規則**：矩陣相乘並非對應位置相乘，而是第一個矩陣的「列」與第二個矩陣的「行」進行內積。
* **索引對應**：在陣列 `m[4]` 中，我們預設 `m[0], m[1]` 為第一列，`m[2], m[3]` 為第二列。
* **運算公式**：
    * 新矩陣 [0] = (左[0]*右[0]) + (左[1]*右[2])
    * 新矩陣 [1] = (左[0]*右[1]) + (左[1]*右[3])
    * 新矩陣 [2] = (左[2]*右[0]) + (左[3]*右[2])
    * 新矩陣 [3] = (左[2]*右[1]) + (左[3]*右[3])

### 2. 運算子多載 (`operator*`)
* 透過自定義 `*` 號，讓 `m3 = m1 * m2` 語法生效，將複雜的矩陣算法封裝在類別內部，使 `main` 函式維持極高的可讀性。

### 3. 輸出格式控制
* 為了符合矩陣視覺效果，在 `show()` 函式中使用 `endl` 換行，將一維陣列呈現為 2x2 的二維方陣格式。

---

## 💻 完整程式碼實作與註解牆

```cpp
#include<iostream>
using namespace std;

class matrix {
private:
    int m[4];                                               // 使用一維陣列儲存 2x2 矩陣 (0,1 為首列；2,3 為末列)

public:
    // 設定矩陣數值
    void set(int a, int b, int c, int d) {                  // 接收四個參數並賦值給內部成員
        m[0] = a;                                           // 第一列左元素
        m[1] = b;                                           // 第一列右元素
        m[2] = c;                                           // 第二列左元素
        m[3] = d;                                           // 第二列右元素
    }

    // 【運算子多載】：定義矩陣相乘 (*) 的行為
    matrix operator*(matrix m2) {
        matrix m3;                                          // 建立結果矩陣物件
        
        // 矩陣乘法核心公式：列(Row) x 行(Column)
        m3.m[0] = (m[0] * m2.m[0]) + (m[1] * m2.m[2]);      // 計算結果左上角元素
        m3.m[1] = (m[0] * m2.m[1]) + (m[1] * m2.m[3]);      // 計算結果右上角元素
        m3.m[2] = (m[2] * m2.m[0]) + (m[3] * m2.m[2]);      // 計算結果左下角元素
        m3.m[3] = (m[2] * m2.m[1]) + (m[3] * m2.m[3]);      // 計算結果右下角元素
        
        return m3;                                          // 回傳運算後的矩陣物件
    }

    // 顯示矩陣內容
    void show() {
        cout << "matrix 3:" << endl;                        // 輸出標題
        cout << m[0] << " " << m[1] << endl;                // 輸出第一列
        cout << m[2] << " " << m[3] << endl;                // 輸出第二列
    }
};

int main() {
    int n[8];                                               // 建立長度 8 的緩衝陣列接收輸入
    matrix m1, m2, m3;                                      // 宣告三個矩陣物件
    
    for (int i = 0; i < 8; i++) {
        cin >> n[i];                                        // 批次讀取矩陣 1 與 矩陣 2 的數值
    }
    
    m1.set(n[0], n[1], n[2], n[3]);                         // 初始化矩陣 1
    m2.set(n[4], n[5], n[6], n[7]);                         // 初始化矩陣 2
    
    m3 = m1 * m2;                                           // 執行矩陣乘法 (觸發 operator* )
    
    m3.show();                                              // 輸出運算結果
    
    return 0;                                               // 程式結束
}
