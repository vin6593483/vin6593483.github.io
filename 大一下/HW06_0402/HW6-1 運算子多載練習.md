# 🎓 HW6-1：運算子多載 (Operator Overloading) - 圓形面積比較系統

## 📝 實作重點與解題思路

本次實作的核心在於建立一個 `circle` 類別，並透過 **運算子多載** 技術自定義大於符號 (`>`) 的行為，使其能夠比較兩個圓形的面積，甚至能讓整數直接與物件進行比較。

### 1. 屬性封裝與計算
* **私有屬性**：將半徑 (`bj`)、名稱 (`name`) 與面積 (`area`) 設為 `private`，確保資料安全性。
* **自動計算**：在建構子初始化時直接根據公式 $Area = r^2 \times 3.14$ 計算面積，並儲存於物件中。

### 2. 運算子多載：友元函式 (Friend Function)
* **物件間比較**：實作 `bool operator>(circle c1, circle c2)`，讓主程式能直觀比較兩個圓形物件。
* **常數與物件比較**：為了支援 `100 > c1` 這種常數在左側的語法，必須定義非成員函式的運算子多載。透過 `friend` 關鍵字，讓該函式能存取類別內的私有成員 `area`。

---

## 💻 完整程式碼實作與註解牆

```cpp
#include<iostream>
#include<string>
using namespace std;

class circle {
private:
    float bj;                                               // 私有屬性：半徑
    string name;                                            // 私有屬性：圓形 ID (名稱)
    float area;                                             // 私有屬性：面積

public:
    circle() {                                              // 預設建構子

    }
    
    circle(string a, float b) {                             // 參數建構子：初始化資料並計算面積
        name = a;                                           // 設定圓形名稱
        bj = b;                                             // 設定半徑
        area = bj * bj * 3.14;                              // 依照題目要求，pi 以 3.14 計算面積
    }

    void show() {                                           // 顯示函式：輸出圓形 ID 與面積
        cout << "Circle " << name << ", Area=" << area << endl;
    }

    // 宣告友元函式 (Friend Functions)，使其能存取 private 成員變數 area
    friend bool operator>(circle c1, circle c2);            // 運算子多載：比較兩個圓形物件
    friend bool operator>(int n, circle c);                 // 運算子多載：比較常數與圓形物件
};

// 【運算子多載實作】：物件 vs 物件
bool operator>(circle c1, circle c2) {
    return c1.area > c2.area;                               // 回傳兩圓面積比較結果(布林值)
}

// 【運算子多載實作】：常數 vs 物件
bool operator>(int n, circle c) {
    return n > c.area;                                      // 回傳常數與圓面積比較結果(布林值)
}

int main(){
    string c11, c22;                                        // 宣告儲存 ID 的字串變數
    float c111, c222;                                       // 宣告儲存半徑的浮點數變數
    
    cin >> c11 >> c111 >> c22 >> c222;                      // 讀取兩組 ID 與半徑輸入
    circle c1(c11, c111), c2(c22, c222);                    // 實體化兩個圓形物件

    // 比較圓 1 與 圓 2 的面積大小
    if (c1 > c2){                                           // 若圓 1 大於 圓 2
        c1.show();                                          // 顯示圓 1 資訊
        cout << "bigger than" << endl;                      // 輸出比較文字
        c2.show();                                          // 顯示圓 2 資訊
    }
    else{                                                   // 若圓 1 不大於 圓 2
        c1.show();
        cout << "smaller than" << endl;
        c2.show();
    }

    // 比較常數 100 與 圓 1 的面積大小
    if (100 > c1){                                          // 觸發友元函式 operator>(int, circle)
        cout << "\n";                                       // 依照範例格式換行
        c1.show();
        cout << "smaller than " << 100 << endl;             // 輸出圓 1 小於 100 的結果
    }
    else{
        cout << "\n";
        c1.show();
        cout << "bigger than " << 100 << endl;              // 輸出圓 1 大於 100 的結果
    }

    return 0;                                               // 程式結束
}
