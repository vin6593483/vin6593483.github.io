# 📝 小考 (03/26) - 多重繼承：相機組件價格計算系統

## 💡 實作重點與解題思路

本次小考要求建立一個能夠整合多個零件價格的相機系統，核心在於實作多重繼承與結構化資料檢索。

### 1. 結構體與資料封裝
* **`Item` 結構體**：定義了品牌 (`brand`) 與對應價格 (`cost`) 的組合。
* **組件類別**：`Lens` (鏡頭) 與 `Tripod` (腳架) 類別各自維護一個私有的零件清單陣列。

### 2. 多重繼承 (Multiple Inheritance)
* **類別整合**：`class mycamera : public Lens, public Tripod`。
* **優點**：`mycamera` 同時繼承了兩個父類別的公開成員函式，使得主程式可以透過單一介面計算整合後的總價。

### 3. 資料搜尋邏輯
* 使用 `find_price` 函式搭配 `for` 迴圈遍歷陣列。
* 透過字串比對 (`a == m[i].brand`) 找出對應品牌的價格，若無匹配則回傳 0。

---

## 💻 完整程式碼實作

```cpp
#include<iostream>
#include<string>
using namespace std;

struct Item {                                               // 定義零件結構體
    string brand;                                           // 品牌名稱
    int cost;                                               // 零件價格
};

class Lens {                                                // 鏡頭基底類別
private:
    Item m[3] = { {"Sony",300},{"Nikon",400},{"Canon",500} };// 鏡頭零件清單
    int out;                                                // 內部狀態變數
public:
    int find_price(string);                                 // 搜尋鏡頭價格介面
};

class Tripod {                                              // 腳架基底類別
private:
    Item c[3] = { {"Manfrotto",100},{"Joby",200},{"Recsur",300} }; // 腳架零件清單
    int out;                                                // 內部狀態變數
public:
    int find_price(string);                                 // 搜尋腳架價格介面
};

// --- 成員函式實作 ---

int Lens::find_price(string a) {                            // 實作鏡頭價格檢索
    for (int i = 0; i < 3; i++) {                           // 遍歷鏡頭清單
        if (a == m[i].brand) {                              // 字串比對成功
            return m[i].cost;                               // 回傳對應價格
        }
    }
    return 0;                                               // 未找到則回傳 0
}

int Tripod::find_price(string b) {                          // 實作腳架價格檢索
    for (int i = 0; i < 3; i++) {                           // 遍歷腳架清單
        if (b == c[i].brand) {                              // 字串比對成功
            return c[i].cost;                               // 回傳對應價格
        }
    }
    return 0;                                               // 未找到則回傳 0
}

// --- 多重繼承衍生類別 ---

class mycamera : public Lens, public Tripod {               // 同時繼承兩個零件類別
private:
    int cost;                                               // 儲存組合總價格
public:
    mycamera(string a, string b) {                          // 建構子：執行整合計算
        cost = Lens::find_price(a) + Tripod::find_price(b); // 加總鏡頭與腳架價格
    }
    void show() {                                           // 顯示結果方法
        cout << "Cost:" << cost;                            // 輸出格式化結果
    }
};

int main() {
    string a, b;                                            // 宣告輸入字串
    cin >> a >> b;                                          // 讀取鏡頭與腳架品牌
    mycamera c1(a, b);                                      // 建立相機物件
    c1.show();                                              // 執行顯示
    return 0;                                               // 程式結束
}
