# 📝 小考 (03/19) - 繼承 & 多載：產品加購折扣系統
1
## 💡 實作重點與解題思路

本次小考要求建立一個繼承自父類別 `product` 的子類別 `buy`，並實作產品價格計算與折扣邏輯。

### 1. 繼承架構與屬性保護
* **父類別 `product`**：定義了受保護屬性 (`protected`) `option` (產品名稱) 與 `add` (是否加購)。
* **權限控制**：使用 `protected` 確保子類別 `buy` 可以自由存取這些資料，但 `main` 函式無法直接修改，維持良好的封裝性。

### 2. 建構元多載 (Constructor Overloading)
為了因應使用者輸入的變數數量不同（有無加購），本實作採用了建構元多載技術：
* **無加購情境 (`N`)**：啟動雙參數建構元，僅計算主產品售價。
* **有加購情境 (`Y`)**：啟動三參數建構元，處理複雜的折扣邏輯。
    * **加購 A 產品**：總金額打 8 折。
    * **加購 B 產品**：總金額打 9 折。

## 💻 完整程式碼實作

```cpp
#include<iostream>
using namespace std;

class product {
protected:
    char option;                                            // 儲存主產品名稱 (A/B)
    char add;                                               // 儲存加購意願 (Y/N)
};

class buy : protected product {
public:
    int sum;                                                // 最終總金額

    // 【建構元多載 A】：處理「有加購」的情境 (傳入 3 個參數)
    buy(char x, char y, char z) {
        option = x;                                         // 初始化主產品名稱
        add = y;                                            // 初始化加購意願
        
        if (x == 'a' || x == 'A') {                         // 如果主產品是 A (100元)
            if (z == 'a' || z == 'A') {
                sum = 160;                                  // 加購A：(100+100)*0.8 = 160
            }
            else {
                sum = 135;                                  // 加購B：(100+50)*0.9 = 135
            }
        }
        else {                                              // 如果主產品是 B (50元)
            if (z == 'a' || z == 'A') {
                sum = 120;                                  // 加購A：(50+100)*0.8 = 120
            }
            else {
                sum = 90;                                   // 加購B：(50+50)*0.9 = 90
            }
        }
    }

    // 【建構元多載 B】：處理「無加購」的情境 (傳入 2 個參數)
    buy(char x, char y) {
        option = x;
        add = y;
        if (x == 'a' || x == 'A') {
            sum = 100;                                      // 僅 A 產品原價
        }
        else {
            sum = 50;                                       // 僅 B 產品原價
        }
    }

    void show() {
        cout << "購買產品:" << option << endl;
        cout << "是否加購:" << add << endl;
        cout << "總金額:" << sum << endl;
    }
};

int main() {
    char p1, p2;                                            // p1:產品, p2:是否加購
    char b;                                                 // b:加購品名稱
    
    cin >> p1 >> p2;                                        // 輸入初始資料

    if (p2 == 'Y' || p2 == 'y') {                           // 判定是否進入加購流程
        cin >> b;                                           // 讀取加購品
        buy b1(p1, p2, b);                                  // 呼叫三個輸入的buy函數
        b1.show();
    }
    else if (p2 == 'N' || p2 == 'n') {
        buy b2(p1, p2);                                     // 呼叫兩個輸入的buy函數
        b2.show();
    }

    return 0;                                               // 程式結束
}
