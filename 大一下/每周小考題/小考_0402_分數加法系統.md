# 📝 小考實作 (04/02) - 運算子多載：同分母分數加法系統

## 💡 實作重點與解題思路

本次小考要求建立一個分數類別 `fraction`，透過 **運算子多載** 實作同分母分數相加，並根據結果格式化輸出為帶分數、真分數或整數。

### 1. 資料結構與進位邏輯
* **屬性定義**：包含分子 (`up`)、分母 (`down`) 與整數部分 (`integer`)。
* **自動轉換**：在建構子與加法運算中，若分子大於等於分母，則執行 `integer++` 並修正分子，將假分數轉為帶分數格式。

### 2. 運算子多載 (`operator+`)
* **空建構子的必要性**：為了在 `operator+` 函式中建立暫存物件 `fraction f`，類別必須提供一個不帶參數的預設建構子。
* **累加運算**：將兩個分數的分子與整數部分各自相加，分母保持不變。

### 3. 多重輸出判定
* 根據運算結果，`show()` 方法需處理三種情況：
    1. **純分數**：整數部分為 0。
    2. **純整數**：分子部分為 0（整除）。
    3. **帶分數**：同時具備非零的整數與分子部分。

---

## 💻 完整程式碼實作與註解牆

```cpp
#include<iostream>
using namespace std;

class fraction {
private:
    int up;                                                 // 私有屬性：分子
    int down;                                               // 私有屬性：分母
    int integer;                                            // 私有屬性：帶分數的整數部分

public:
    // 預設建構子：支援運算子多載內部建立臨時物件
    fraction() {                                            // 若無此空建構元，operator+ 內的宣告會報錯
        // 保留空白即可
    }

    // 參數建構子：初始化並自動執行假分數轉換
    fraction(int a, int b) {
        up = a;                                             // 設定分子
        down = b;                                           // 設定分母
        integer = 0;                                        // 初始化整數部分為 0
        
        if (up >= down) {                                   // 判定是否為假分數
            integer++;                                      // 整數部分加 1
            up = up - down;                                 // 分子減去分母（處理簡單進位）
        }
    }

    // 【運算子多載】：定義兩個分數物件相加 (+) 的行為
    fraction operator+(fraction f2) {
        fraction f;                                         // 建立結果物件（依賴空建構子）
        f.up = up + f2.up;                                  // 分子相加
        f.down = down;                                      // 分母維持相同
        f.integer = integer + f2.integer;                   // 整數部分相加
        
        if (f.up >= f.down) {                               // 運算後再次檢查是否進位
            f.integer++;                                    // 進位至整數部分
            f.up = f.up - f.down;                           // 修正分子
        }
        return f;                                           // 回傳計算完畢的分數物件
    }

    // 格式化顯示結果
    void show() {
        if (integer == 0) {                                 // 情況 1：結果為真分數
            cout << up << "/" << down << endl;
        }
        else if (up == 0) {                                 // 情況 2：結果為整除後的整數
            cout << integer << endl;
        }
        else {                                              // 情況 3：結果為帶分數
            cout << integer << " " << up << "/" << down << endl;
        }
    }
};

int main() {
    int Numerator_1, Numerator_2, Denominator;
    
    // 讀取輸入：分子1, 分子2, 共同分母
    cin >> Numerator_1 >> Numerator_2 >> Denominator;
    
    fraction f1(Numerator_1, Denominator);                  // 實體化分數 1
    fraction f2(Numerator_2, Denominator);                  // 實體化分數 2
    
    fraction f3 = f1 + f2;                                  // 執行加法 (觸發 operator+)
    
    f3.show();                                              // 輸出最終結果
    
    return 0;                                               // 程式結束
}
