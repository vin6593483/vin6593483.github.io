# HW3-1 繼承與多型 (Inheritance & Polymorphism)

## 📝 實作重點與解題思路

本次實作的核心在於練習 **C++ 的類別繼承** 以及 **函式多載 (Overloading)** 的應用。題目已固定了父類別 `person` 以及 `main` 主程式，我們只需專注於實作子類別 `student` 的內部邏輯。

### 1. 類別繼承
我們透過 `class student:public person` 的語法，讓 `student` 類別繼承 `person` 類別的所有公開 (public) 屬性與方法。
* **封裝性考量**：由於父類別的 `ID` 與 `NAME` 是 `private`，子類別無法直接存取。因此在 `student` 的建構子中，我們必須呼叫父類別提供的 `set_id(id)` 與 `set_name(name)` 來完成初始化。

### 2. 屬性&方法
根據題目要求，`student` 類別新增了三個專屬的浮點數 (float) 屬性來儲存成績：
* `chinese` (中文)
* `math` (數學)
* `english` (英文)

並且實作了以下方法：
* `set_score(float a, float b, float c)`：將傳入的三個參數分別指定給對應的科目變數。
* `show(void)`：依序輸出中、數、英三科的成績。

### 3. 多型
為了處理「原始平均」與「加分後平均」兩種不同情境，我們使用了函式多載的技巧，宣告了兩個同名但參數不同的 `average` 函式：
* **`float average(void)`**：
  不接收任何參數，單純計算並回傳三科的基礎平均分數。
* **`float average(float add)`**：
  接收一個浮點數參數 `add` 代表額外加分。在計算基礎平均後加上 `add`。為了符合題目要求，特別加入了條件判斷 `if (new_avg > 100)`，確保加分後的總平均最高只會落在 **100分**。

---

## 💻 程式碼架構分析

```cpp
class student:public person {
private:
	float chinese;
	float math;
	float english;
public:
	// 建構子：利用父類別的 setter 完成初始化
	student(int id,string name) {
		set_id(id);
		set_name(name);
	}
	
	// 設定成績
	void set_score(float a, float b, float c) {
		chinese = a;
		math = b;
		english = c;
	}
	
	// 顯示各科成績
	void show(void) {
		cout << "中文:" << chinese << endl;
		cout << "數學:" << math << endl;
		cout << "英文:" << english << endl;
	}
	
	// 回傳原始平均
	float average(void) {
		float avg = (chinese + math + english) / 3;
		return avg;
	}
	
	// 回傳加分後的平均 (函式多載)
	float average(float add) {
		float new_avg = ((chinese + math + english) / 3) + add;
		// 確保最高分不超過 100
		if (new_avg > 100) {
			new_avg = 100;
		}
		return new_avg;
	}
};
