# 🎓 HW4-1：多重繼承 (Multiple Inheritance) - 組裝電腦成本計算系統

## 📝 實作重點與解題思路

本次實作的核心在於練習 **C++ 的多重繼承**，模擬現實中組裝電腦需要整合多種硬體組件的過程。

### 1. 多重繼承架構
* **獨立組件類別**：分別建立了 `Monitor` (螢幕)、`CPU` (處理器) 與 `RAM` (記憶體) 三個基底類別。
* **多重複合繼承**：`class mycomp : public Monitor, public CPU, public RAM`。
    * 透過此語法，`mycomp` 類別同時擁有了三個父類別的所有公開方法，能直接呼叫各組件的價格查詢功能。

### 2. 結構體 (Struct) 與資料檢索
* 使用 `struct Item` 定義品牌 (`Brand`) 與價格 (`price`) 的對應關係。
* 各類別內部維護一個長度為 3 的 `Item` 陣列（如 Sony, ViewSonic, Samsung）。
* 實作 `find_price` 函式，透過 `for` 迴圈與字串比對，精確找出對應品牌的價格。

### 3. 建構子與作用域解析 (Scope Resolution)
* **多載建構子**：提供預設建構子（歸零）與參數建構子（計算總價）。
* **作用域指定**：由於三個父類別都有同名的 `find_price` 方法，在子類別中使用 `Monitor::find_price()` 這種語法能明確告訴編譯器要執行哪一個父類別的函式，有效避免命名衝突。

## 💻 完整程式碼實作與註解牆

```cpp
#include<iostream>
#include<string>
using namespace std;

struct Item {                                               // 定義硬體項目的結構體
    string Brand;                                           // 儲存品牌名稱
    int price;                                              // 儲存對應價格
};

class Monitor {                                             // 螢幕類別
private:
    Item m[3] = { {"Sony",100},{"ViewSonic",200},{"Samsung",300} };
public:
    int find_price(string);                                 // 搜尋螢幕價格
};

class CPU {                                                 // 處理器類別
private:
    Item c[3] = { {"Core_i3",100},{"Core_i5",200},{"Core_i7",300} };
public:
    int find_price(string);                                 // 搜尋處理器價格
};

class RAM {                                                 // 記憶體類別
private:
    Item r[3] = { {"Kingston",100},{"ADATA",200},{"Micron",300} };
public:
    int find_price(string);                                 // 搜尋記憶體價格
};

// --- 以下為各類別成員函式的實作 ---

int Monitor::find_price(string a) {
    for (int i = 0; i < 3; i++) {
        if (m[i].Brand == a) return m[i].price;             // 比對品牌並回傳螢幕價格
    }
    return 0;
}

int CPU::find_price(string b) {
    for (int i = 0; i < 3; i++) {
        if (c[i].Brand == b) return c[i].price;             // 比對品牌並回傳處理器價格
    }
    return 0;
}

int RAM::find_price(string c) {
    for (int i = 0; i < 3; i++) {
        if (r[i].Brand == c) return r[i].price;             // 比對品牌並回傳記憶體價格
    }
    return 0;
}

// --- 多重繼承核心類別 ---

class mycomp : public Monitor, public CPU, public RAM {     // 同時繼承螢幕、CPU與RAM類別
private:
    int cost;                                               // 儲存組裝總成本
public:
    mycomp() { cost = 0; }                                  // 預設建構子
    mycomp(string a, string b, string c) {                  // 參數建構子：執行組裝計算
        // 使用作用域運算子呼叫特定父類別的同名函式
        cost = Monitor::find_price(a) + CPU::find_price(b) + RAM::find_price(c);
    }
    void show() {
        cout << "Cost:" << cost << endl;                    // 輸出最終計算結果
    }
};

int main() {
    string a, b, c;                                         // 定義三個輸入字串
    cin >> a >> b >> c;                                     // 讀取螢幕、處理器、記憶體品牌
    mycomp c1(a, b, c);                                     // 建立電腦物件並計算價格
    c1.show();                                              // 顯示成本
    return 0;                                               // 程式結束
}
