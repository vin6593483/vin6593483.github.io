# 作業 7-4：雙開關電源控制

這題是非常經典的數位邏輯應用，模擬生活中常見的「雙向開關」（例如樓梯上下各有一個開關控燈）。程式核心利用了位元運算子 `^` (XOR)，當兩個輸入狀態不同時回傳 1 (亮燈)，狀態相同則回傳 0 (熄燈)。同時透過 `void` 函數處理輸出，讓程式架構層次分明。

## 💡 程式碼與解題思路

詳細的 XOR 邏輯與自訂函數的用法都整理在右側的註解牆：

```cpp
#include<iostream>
using namespace std;
int rt_light_state(int a, int b) {                       // 自訂函數：計算燈泡狀態，回傳整數值
	int i = a ^ b;                                       // 核心邏輯：使用 XOR (互斥或) 運算。相同為 0，不同為 1
	return i;                                            // 回傳計算結果
}
void Light_ONorOFF(int n) {                              // 自訂函數：根據輸入值判斷並輸出文字，無回傳值 (void)
	if (n == 1) {                                        // 如果狀態值為 1
		cout << "light On\n";                            // 輸出燈亮訊息並換行
	}
	else if (n == 0) {                                   // 如果狀態值為 0
		cout << "light Off\n";                           // 輸出燈滅訊息並換行
	}
	else {                                               // 若輸入非 0 也非 1 (防呆機制)
		cout << "error!!\n";                             // 輸出錯誤訊息
	}
}
int main(void) {                                         // 進入主程式
	int bt_A, bt_B, light_result;                        // 宣告開關 A、開關 B 以及結果變數
	cin >> bt_A >> bt_B;                                 // 讀取兩個開關的狀態 (0 或 1)
	light_result = rt_light_state(bt_A, bt_B);           // 呼叫函數進行 XOR 運算，取得燈泡應有的狀態
	Light_ONorOFF(light_result);                         // 將運算結果傳給輸出函數，顯示最終燈號

	return 0;                                            // 程式順利結束
}
```
