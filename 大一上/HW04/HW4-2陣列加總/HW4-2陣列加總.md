# 作業 4-2：陣列加總

這題主要練習一維陣列 (`array`) 的基本操作，以及如何搭配 `for` 迴圈來進行特定範圍的資料讀取與局部加總，最後再把兩位業務員的業績合併計算。

## 💡 程式碼與解題思路

我把註解統一整理在畫面最右側，同學可以直接垂直閱讀程式碼的核心邏輯，不會被打斷：

```cpp
#include <iostream>
using namespace std;
int main(void) {
    int sales[8];                                                                                          // 宣告長度為 8 的整數陣列，儲存兩位業務的業績
    for (int i = 0; i < 8; i++) {                                                                          // 利用迴圈連續讀取 8 筆資料
        cin >> sales[i];                                                                                   // 將輸入的值依序存入陣列
    }
    int total1 = 0;                                                                                        // 宣告變數 total1 儲存第一位業務的總業績
    for (int i = 0; i < 4; i++) {                                                                          // 讀取陣列前 4 個元素 (索引 0 到 3)
        total1 = total1 + sales[i];                                                                        // 將第一位業務的業績加總
    }
    int total2 = 0;                                                                                        // 宣告變數 total2 儲存第二位業務的總業績
    for (int i = 4; i < 8; i++) {                                                                          // 讀取陣列後 4 個元素 (索引 4 到 7)
        total2 = total2 + sales[i];                                                                        // 將第二位業務的業績加總
    }
    cout << "salesman1:" << sales[0] << "," << sales[1] << "," << sales[2] << "," << sales[3] << "\n";     // 輸出第一位業務的各筆業績並以逗號分隔
    cout << "total:" << total1 << "\n";                                                                    // 輸出第一位業務的總業績
    cout << "salesman2:" << sales[4] << "," << sales[5] << "," << sales[6] << "," << sales[7] << "\n";     // 輸出第二位業務的各筆業績並以逗號分隔
    cout << "total:" << total2 << "\n";                                                                    // 輸出第二位業務的總業績
    cout << "sum:" << total1 + total2 << "\n";                                                             // 輸出兩位業務員的業績總和
    return 0;                                                                                              // 程式順利結束，釋放記憶體
}
```
