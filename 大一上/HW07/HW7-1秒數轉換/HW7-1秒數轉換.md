# 作業 7-1：秒數轉換

這題主要練習數學運算子中的 `/` (取商) 與 `%` (取餘數) 的實際應用。透過除以 3600 和 60，我們可以將一長串的總秒數，精準地轉換成「時：分：秒」的格式。同時也加入了防呆機制，排除負數與超過一天的異常數值。

## 💡 程式碼與解題思路

時間轉換的數學邏輯與防呆機制，我都詳細寫在右側的註解裡了：

```cpp
#include<iostream>
using namespace std;
int main(void) {
	int s;                                          // 宣告變數 s，用來儲存使用者輸入的總秒數
	cin >> s;                                       // 讀取輸入的秒數
	if (s >= 86400 || s<0) {                        // 防呆機制：一天有 86400 秒，若超過一天或輸入負數
		cout << "error!\n";                         // 則輸出 error! (錯誤訊息)
		return 0;                                   // 提早結束程式 (Early Return)，不繼續往下算
	}
	int a, b, c;                                    // 宣告變數 a(時)、b(分)、c(秒)
	a = (s / 3600);                                 // 1小時=3600秒，總秒數除以3600得到「小時」
	b = (s % 3600) / 60;                            // 取除以3600的餘數(剩下不到1小時的秒數)，再除以60得到「分鐘」
	c = (s % 3600) % 60;                            // 剩下的秒數再對60取餘數，就是最後剩下的「秒數」
	cout << a << ":" << b << ":" << c;              // 依題目要求的格式 a:b:c (時:分:秒) 輸出結果
	return 0;                                       // 程式順利結束，釋放記憶體
}
```
