# 作業 7-3：指定字元檢索

這題練習字元陣列 (`char array`) 的字串輸入與檢索。程式會讓使用者輸入一串文字以及一個指定的目標字元，然後利用 `for` 迴圈逐一比對陣列中的每一個位置，並計算目標字元總共出現的次數。

## 💡 程式碼與解題思路

簡單暴力的「陣列尋寶」邏輯，詳細的變數解說與溫馨小提醒都寫在右側的註解裡了：

```cpp
#include<iostream>
using namespace std;
int main(void) {
	char a[20];                                         // 宣告長度為 20 的字元陣列 a，用來儲存輸入的字串
	char b;                                             // 宣告字元變數 b，用來儲存想要檢索的「目標字元」
	cin >> a >> b;                                      // 依序讀取使用者輸入的字串與目標字元
	int count = 0;                                      // 宣告計數器 count，預設為 0，用來記錄出現的次數
	for (int i = 0; i <= 19; i++) {                     // 跑迴圈檢查。(偷偷提醒：大小20的陣列索引最多到19，寫 <=20 會讀出界喔！)
		if (a[i] == b) {                                // 如果陣列當前位置的字元，剛好等於我們要找的目標字元 b
			count++;                                    // 就把計數器加 1
		}
	}
	cout << count;                                      // 迴圈結束後，輸出目標字元總共出現的次數
	return 0;                                           // 程式順利結束，釋放記憶體
}
```
