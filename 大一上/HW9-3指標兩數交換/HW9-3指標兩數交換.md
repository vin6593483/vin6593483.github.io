# 作業 9-3：指標兩數交換

這題練習指標最經典的應用——「兩數交換 (Swap)」。透過傳遞變數的記憶體地址，我們可以在自訂函數中直接對主程式的變數進行數值調換，而不僅僅是傳遞數值副本。這是理解 C++ 記憶體操作非常關鍵的一步。

## 💡 程式碼與解題思路

詳細的指標操作與交換邏輯，我都標註在右側註解區：

```cpp
#include<iostream>
using namespace std;
void change_num(int* a, int* b) {                     // 函數接收兩個整數指標（地址），使用 void 因為不需回傳值
	int c = *a;                                      // 宣告暫存變數 c，先把地址 a 指向的數值存起來
	*a = *b;                                         // 將地址 b 指向的數值，覆蓋到地址 a 所指向的空間
	*b = c;                                          // 將暫存變數 c 中的原 a 數值，放到地址 b 所指向的空間
}
int main(void) {                                     // 進入主程式
	int a, b;                                        // 宣告兩個整數變數 a 和 b
	cin >> a >> b;                                   // 讀取使用者輸入的兩個要交換的數字
	change_num(&a, &b);                              // 關鍵：使用 & 運算子將 a 與 b 的「地址」傳入函數進行交換
	cout << a << " " << b << endl;                   // 輸出交換後的結果，確認主程式的變數值已被修改
	return 0;                                        // 程式順利結束
}
```
