# 作業 10-3：副程式處理全班成績

這題練習如何結合「動態記憶體配置」與「副程式 (Function)」來處理班級成績。我們先利用 `malloc` 根據輸入的人數 `n` 申請空間，並將該地址傳入副程式 `pass` 中。副程式利用指標巡視所有成績，統計及格 (>= 60) 的人數並回傳。最後記得釋放記憶體，保持程式的高效與整潔。

## 💡 程式碼與解題思路

副程式的指標操作與主程式的動態管理細節，我都寫在右側註解區：

```cpp
#include<iostream>
using namespace std;
int pass(int n, int* ptr) {                                         // 副程式：統計及格人數，接收人數 n 與成績指標 ptr
	int count = 0;                                                  // 宣告計數器 count 並歸零
	for (int i = 0; i < n; i++) {                                   // 跑迴圈巡視所有動態配置的記憶體空間
		if (*(ptr + i) >= 60) {                                     // 利用指標偏移取得數值，判斷是否及格 (>= 60)
			count++;                                                // 若及格，則計數器加 1
		}
	}
	return count;                                                   // 回傳最終統計的及格總人數
}
int main(void) {                                                    // 進入主程式
	int n;                                                          // 宣告變數 n 儲存全班人數
	int* ptr;                                                       // 宣告整數指標 ptr
	cin >> n;                                                       // 讀取人數
	ptr = (int*)malloc(n * sizeof(int));                            // 核心：向系統申請 n 個整數大小的動態空間
	for (int i = 0; i < n; i++) {                                   // 迴圈讀取全班成績
		cin >> *(ptr + i);                                          // 利用指標偏移依序存入每位同學的分數
	}
	cout << pass(n, ptr) << endl;                                   // 呼叫副程式並傳入地址，直接輸出回傳的人數結果
	free(ptr);                                                      // 重要：程式結束前釋放動態申請的記憶體，避免洩漏
	return 0;                                                       // 程式順利結束
}
```
