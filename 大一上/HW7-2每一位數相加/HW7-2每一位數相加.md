# 作業 7-2：每一位數相加

這題進入了「自訂函數 (Function)」的領域！我們建立了一個獨立的函數來處理計算，讓主程式保持乾淨。核心的數學邏輯是利用 `while` 迴圈，透過 `% 10` 取出數字的最後一位，再用 `/ 10` 像剝洋蔥一樣把最後一位剝除，逐步把每一位數拆解下來加總。

## 💡 程式碼與解題思路

數字拆解的「剝洋蔥」邏輯以及函數的呼叫方式，我都詳細寫在右側的註解裡了：

```cpp
#include<iostream>
using namespace std;
int add_digits(int a) {                                         // 自訂函數：專門用來計算數字每一位的總和
	int sum = 0;                                                // 宣告總和變數 sum 並歸零
	while (a != 0) {                                            // 當數字 a 還沒被除到變成 0 時，繼續執行迴圈
		sum = sum + (a % 10);                                   // 核心邏輯：a % 10 可以精準取出最後一位數字，加進總和裡
		a = a / 10;                                             // 將 a 除以 10，也就是捨去已經加過的最後一位數字
	}
	return sum;                                                 // 迴圈結束後，回傳計算完畢的總和給主程式
}
int main(void) {                                                // 進入主程式
	int n;                                                      // 宣告變數 n 儲存使用者輸入的數字
	cin >> n;                                                   // 讀取輸入的數字
	if (n < 1000000000) {                                       // 防呆機制：確保輸入的數字在 10 億以內 (避免整數溢位)
		cout << add_digits(n) << endl;                          // 呼叫我們上面寫好的 add_digits 函數計算總和，並輸出結果與換行
	}
	return 0;                                                   // 程式順利結束，釋放記憶體
}
```
