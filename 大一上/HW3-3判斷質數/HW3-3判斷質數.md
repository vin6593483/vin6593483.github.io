# 作業 3-3：判斷質數

這題的重點在於判斷一個數字是否為質數。除了處理 1 以下的例外狀況，利用 `for` 迴圈從 2 開始除到 `a-1`，只要一找到因數就立刻中斷程式，是非常有效率的做法。

## 💡 程式碼與解題思路

程式碼內部有詳細的註解說明，特別是提早結束迴圈的邏輯：

```cpp
#include<iostream>
using namespace std;
int main(void) {
	int a;
	cin >> a;                               //讓使用者輸入要判斷的數
	if (a <= 1) {                           //先判斷前置條件:負數、0、1不是質數
		cout << "Not prime number\n";
	}
	int i;                                  //設立for迴圈運算值
	for (i = 2; i < a; i++) {               //for(從2開始;到a的前一個數結束;i每走完一圈+1)
		if (a % i == 0) {                   //在2~(a-1)如果能找到一個數，跟a取餘數是0(即有因數)，則直接輸出"不是質數"(因為找到有一個因數就夠了，題目沒要求列舉出來，於是只要找到一個因數就代表不是質數，可以直接輸出結果不用等全部找完)
			cout << "Not prime number\n";   //為什麼要取到a-1的原因是:如果結束值取到a，a本身必定能讓餘數為0，程式就會輸出說"不是質數"，所以不能讓for迴圈運行到a
			return 0;                       //重設所有記憶體(因為已經找到了，知道不是質數了，沒有運行下去的必要遂直接停止)
		}
	}
	cout << "Prime number\n";               //找不到其餘因數，則輸出"是質數"
	return 0;                               //重設所有記憶體
}
