# 作業 10-4：氣泡排序排列

這題是演算法的經典入門——「氣泡排序法 (Bubble Sort)」。我們結合了動態記憶體配置來儲存資料，並透過副程式進行排序。核心邏輯在於雙層迴圈：外層控制掃描次數，內層則不斷比較相鄰的兩個數字，若順序不對就進行交換 (Swap)，直到所有數字都按順序排列。

## 💡 程式碼與解題思路

詳細的排序邏輯與指標操作細節，我都對齊在右側註解牆供大家參考：

```cpp
#include<iostream>
#include<cstdlib>
using namespace std;
void sort(int* arr,int n) {                                         // 副程式：氣泡排序，接收陣列位址與數量 n
	for (int i = 0; i < n-1 ; i++) {                                // 外層迴圈：控制總共要掃描幾輪
		for (int j = 0; j < n - 1 - i; j++) {                        // 內層迴圈：比較相鄰元素，每一輪會把最大值推到最後面
			if (arr[j] > arr[j + 1]) {                              // 如果前面的數字比後面大
				int temp = arr[j];                                  // 經典交換 (Swap) 邏輯：利用暫存變數 temp
				arr[j] = arr[j + 1];                                // 將後面的數字換到前面
				arr[j + 1] = temp;                                  // 將 temp 裡存的前面數字換到後面
			}
		}
	}
}
int main(void) {                                                    // 進入主程式
	int n;                                                          // 宣告變數 n 儲存要排序的數字數量
	cin >> n;                                                       // 讀取數量
	int* arr;                                                       // 宣告整數指標 arr
	arr = (int*)malloc(n*sizeof(int));                                
	for (int i = 0; i < n; i++) {                                   // 迴圈讀取輸入資料
		cin >> arr[i];                                              // 將輸入的數字依序存入動態配置的空間
	}
	sort(arr,n);                                                    // 呼叫排序副程式，傳入地址進行「隔山打牛」的調換
	for (int i = 0; i < n; i++) {                                   // 迴圈輸出排序後的結果
		cout << arr[i] << " ";                                      // 依序輸出並以空格分隔
	}
	return 0;                                                       // 程式順利結束
}
```
