# 作業 6-1：學生成績統整系統

這題進入了 `struct` (結構) 的領域！透過自訂的資料型態，我們可以把學生的「性別」和「分數」綁定在一起處理。程式核心邏輯是利用陣列儲存 5 筆結構資料，再透過迴圈與條件判斷，分別計算出男、女生的平均分數。

## 💡 程式碼與解題思路

註解裡滿滿的重點（還有單押的詩句），大家可以直接看右側的白話文解說：

```cpp
#include<iostream>
using namespace std;
struct student {                                                                       // 依題目要求，先建立一個struct(我命名為student)
	char gender;                                                                       // 建立一欄性別，因為是字符用char形式
	float score;                                                                       // 接下來是分數欄，因為有可能有小數所以選擇float，double尤佳
};
int main(void) {                                                                       // (一陣強烈的電吉他)進入主程式!
	student s[5];                                                                      // 首先先宣告建立一個陣列，以student結構保存，裡頭存放5筆資料
	for (int i = 0; i < 5; i++) {                                                      // 這個for迴圈是為了讓使用者輸入這5筆student結構的資料
		cin >> s[i].gender >> s[i].score;                                              // 想要重複不費力，選for不會辜負你
	}                                                                                  // 思路：每個性別叫兩塊記憶體，一塊負責計人數，一塊負責加總分數
	float boysum = 0, girlsum = 0;                                                     // 於是我這邊先宣告了float型態的計總和的記憶體
	int boycount = 0, girlcount = 0;                                                   // 再來是整數形式的計人數的記憶體
	for (int i = 0; i < 5; i++) {                                                      // 接下來因為也要重複作業(判斷男女)，所以再次寵幸for迴圈
		if (s[i].gender == 'b') {                                                      // 當發現是小男孩的時候就在計人數那邊+1，同時加總分數
			boycount++;                                                                // 註：單引號是因為char，按ASCII/Unicode儲存(041教過，佔1byte)
			boysum = boysum + s[i].score;                                              // 註：雙引號""則是按string(字串)輸出。簡單理解：單引號char，雙引號string
		}
		else if (s[i].gender == 'g') {                                                 // 女生也做一次一樣的統計
			girlcount++;
			girlsum = girlsum + s[i].score;
		}
	}
	cout << (boysum / boycount) << "\n";                                               // 最後直接輸出男生的平均 (總和/人數)就好
	cout << (girlsum / girlcount) << "\n";                                             // 輸出女生的平均
	return 0;                                                                          // 程式結束！
}
```
