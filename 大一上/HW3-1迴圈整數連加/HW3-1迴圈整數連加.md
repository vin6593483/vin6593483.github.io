# 作業 3-1：迴圈整數連加
這題主要是練習 `for` 迴圈的使用，以及判斷起點和終點的大小關係 (a 是否大於 b)。如果輸入的起點大於終點，程式會自動阻擋並輸出 error!。

## 💡 程式碼與解題思路
程式碼內部有非常詳細的註解，同學可以直接參考我的邏輯：

```cpp
#include<iostream>
using namespace std;
int main(void) {                //綠色的字是註解，電腦碰到會跳過不理他，用來說明或是分段程式內容很有用
	int a, b;
	cin >> a >> b;
	if (a > b) {                //先判斷a是否小於b(就看有沒有大於嘛)，不是的話直接輸出error!
		cout << "error!\n";
		return 0;
	}
	int c, i;                   //前置條件判斷完成(即a<=b)，則開始進行主程式，先說明我的變數是什麼:c是我要的總和值，i是讓for迴圈能運行起來的計算值(就是讓他跑的東西有記憶體可以記)
	c = 0;                      //令c等於0(讓c在算之前是0，等等放啥再說)
	for (i = a; i <= b; i++) {  //for(開始值;結束值;變數)，我讓i從a開始然後一直到b結束，中間變的就是每走完一圈i+1
		c = c + i;              //這行的意思是，每走一圈c都要進行一次運算去覆蓋上一圈的值，每圈的新c都等於舊c加上新的i，假設上一迴圈的c是6，i是8，那這一迴圈的新c=6(舊c)+9(新i)=15
	}
	cout << c;                  //迴圈到加到b結束，最後輸出最新的c的值
	return 0;                   //重設所有記憶體狀態(回到原本情況)
}
