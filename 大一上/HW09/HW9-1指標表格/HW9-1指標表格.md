# 作業 9-1：指標表格

這題練習大規模陣列的初始化與資料存放。透過宣告一個長度為 54 的整數陣列，將特定的數值依序填入每個索引位置，最後利用 `for` 迴圈搭配分隔符號 `|` 將所有數據輸出成一個簡易的表格格式。

## 💡 程式碼與解題思路

雖然輸入量很大，但邏輯很單純，詳細的註解對齊在右側供大家參考：

```cpp
#include<iostream>
using namespace std;
int main() {
    int a[54];                                      // 宣告一個長度為 54 的整數陣列，用來存放大量的指標數據
    a[0] = 28;  a[1] = 16;  a[2] = 2000;            // 以下開始手動為陣列的每個位置賦值
    a[3] = 16;  a[4] = 1000; a[5] = 28;             // 這些數值代表表格中的各項指標資料
    a[6] = 28;  a[7] = 14;  a[8] = 2000;
    a[9] = 14;  a[10] = 1000; a[11] = 28;
    a[12] = 16; a[13] = 14; a[14] = 2000;
    a[15] = 14; a[16] = 1000; a[17] = 16;
    a[18] = 12; a[19] = 14; a[20] = 2000;
    a[21] = 14; a[22] = 1000; a[23] = 12;
    a[24] = 12; a[25] = 14; a[26] = 2000;
    a[27] = 14; a[28] = 2000; a[29] = 14;
    a[30] = 12; a[31] = 19; a[32] = 2000;
    a[33] = 19; a[34] = 2000; a[35] = 19;
    a[36] = 12; a[37] = 19; a[38] = 1000;
    a[39] = 12; a[40] = 2000; a[41] = 19;
    a[42] = 7;  a[43] = 19; a[44] = 1000;
    a[45] = 7;  a[46] = 2000; a[47] = 19;
    a[48] = 7;  a[49] = 7;  a[50] = 1000;
    a[51] = 7;  a[52] = 2000; a[53] = 7;
    
    for (int i = 0; i < 54; i++) {                  // 跑迴圈巡視這 54 個儲存格
        cout << a[i] << "|";                        // 依序輸出每個數字，並在後面補上分隔號 |
    }
    
    return 0;                                       // 程式結束
}
```
