# 作業 9-2：指標兩數相加

這題進入了 C++ 的核心——「指標 (Pointer)」的應用。我們不再只是傳遞數值，而是直接傳遞變數在記憶體中的「地址」。透過在函數參數中使用 `int *` 接收地址，並在運算時加上 `*` (提領算符) 來取出該地址存放的數字進行相加。

## 💡 程式碼與解題思路

指標與地址的對應關係我都詳細寫在右側註解牆了，方便大家垂直閱讀邏輯：

```cpp
#include<iostream>
using namespace std;
int add_num(int *a, int *b) {                        // 函數接收兩個整數指標（即記憶體地址）
	int c = 0;                                       // 宣告變數 c 用來存放相加結果
	c = *a + *b;                                     // 使用 * 符號取出指標 a 與 b 所指向地址裡的「值」並相加
	return c;                                        // 回傳加總結果
}
int main(void) {                                     // 進入主程式
	int a, b;                                        // 宣告兩個整數變數 a 和 b
	cin >> a >> b;                                   // 讀取使用者輸入的兩個數字
	cout << add_num(&a, &b) << endl;                 // 使用 & 取出 a 與 b 的「地址」傳入函數，並輸出結果
	return 0;                                        // 程式順利結束
}
```
