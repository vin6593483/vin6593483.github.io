# 作業 4-3：最大值與最小值

這題練習陣列的進階應用，核心邏輯是先「假設」陣列的第一個數字是最大值與最小值，接著再利用 `for` 迴圈從第二個數字開始往後檢查。只要遇到更大的就換掉最大值，遇到更小的就換掉最小值。

## 💡 程式碼與解題思路

詳細的比較邏輯都寫在右側的註解區塊，大家可以順著迴圈的思路往下看：

```cpp
#include<iostream>
using namespace std;
int main(void) {
	int num[10];                                    // 宣告長度為 10 的整數陣列，用來儲存輸入的數字
	int i;                                          // 宣告迴圈用的計數變數 i
	for (i = 0; i < 10; i++) {                      // 跑 10 次迴圈，依序讀取使用者輸入的值
		cin >> num[i];                              // 將輸入的數字存進陣列的對應位置
	}
	int max = num[0];                               // 先預設陣列的第一個元素 (索引 0) 為目前的「最大值」
	int min = num[0];                               // 同理，也先預設第一個元素為目前的「最小值」
	for (i = 1; i < 10; i++) {                      // 從陣列的第二個元素 (索引 1) 開始往後檢查到最後
		if (num[i] > max) {                         // 如果當前的數字比我們記錄的「最大值」還要大
			max = num[i];                           // 就更新「最大值」為這個新的數字
		}
		if (num[i] < min) {                         // 反之，如果當前的數字比記錄的「最小值」還要小
			min = num[i];                           // 就更新「最小值」為這個新的數字
		}
	}
	cout << "max:" << max << "\n";                  // 迴圈結束後，輸出最終找出的最大值
	cout << "min:" << min << "\n";                  // 接著輸出最終找出的最小值
	return 0;                                       // 程式順利結束，釋放記憶體
}
```
