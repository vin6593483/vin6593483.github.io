# 作業 4-1：成績判斷

這題主要是練習基礎的 `if-else` 條件判斷，以及字元陣列 (`char array`) 的字串輸入。程式會先防呆判斷分數是否超過 100 分（超過則強制以 100 分計算），並根據 60 分的及格門檻輸出 passed 或 failed。

## 💡 程式碼與解題思路

程式邏輯非常直觀，我把變數宣告與判斷條件的細節都寫在右側註解裡：

```cpp
#include<iostream>
using namespace std;
int main(void) {
	int score;                      // 宣告整數變數來儲存分數
	char name[100];                 // 宣告字元陣列來儲存名字，長度設為 100 避免溢位
	cin >> name >> score;           // 讓使用者依序輸入名字和分數
	if (score > 100) {              // 防呆機制：如果輸入的分數超過 100，強制設定為 100
		score = 100;
	}
	cout << "Name:" << name << "\n"; // 輸出學生的名字，並加上換行符號 \n
	cout << "Score:" << score << "\n";// 輸出學生的分數，並加上換行符號 \n
	if (score < 60) {               // 判斷是否及格（60分為及格線），若低於 60 分
		cout << name << " failed."; // 則輸出 failed
	}
	else {                          // 否則（60 分含以上）
		cout << name << " passed."; // 則輸出 passed
	}
	return 0;                       // 程式順利結束，釋放記憶體
}
```
