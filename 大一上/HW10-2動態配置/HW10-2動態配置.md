# 作業 10-2：動態配置與函數應用

這題結合了「動態記憶體配置」與「函數指標傳遞」。我們利用 `malloc` 根據使用者需求的數量申請記憶體，並將該地址傳入兩個獨立函數：一個負責找出最大值，另一個負責計算平均值。最後，我們遵循良好的開發習慣，使用 `free` 歸還記憶體空間。

## 💡 程式碼與解題思路

函數模組化與指標運算的細節，我都詳細對齊在右側註解區了：

```cpp
#include<iostream>
using namespace std;
void MAX(int n, float* ptr) {                                               // 自訂函數：找出最大值，接收資料量與指標地址
	float max = *(ptr);                                                     // 預設第一個元素為最大值
	for (int i = 1; i < n; i++) {                                           // 從第二個元素開始往後比對
		if (*(ptr + i) > max) {                                             // 如果當前指標指向的數值大於紀錄的最大值
			max = *(ptr + i);                                               // 更新最大值
		}
	}
	cout << max << " ";                                                     // 輸出最大值並加上空格
}
void ALL(int n, float* ptr) {                                               // 自訂函數：計算平均值
	float all = 0;                                                          // 宣告累加變數 all
	for (int i = 0; i < n; i++) {                                           // 跑迴圈巡視所有動態配置的記憶體
		all = all + *(ptr + i);                                             // 將數值逐一累加
	}
	float pingjun = all / n;                                                // 計算平均 (總和 / 數量)
	cout << pingjun;                                                        // 輸出平均值
}
int main(void) {                                                            // 進入主程式
	int n;                                                                  // 宣告變數 n 用來儲存資料數量
	cin >> n;                                                               // 讀取使用者輸入的數量
	float* ptr;                                                             // 宣告浮點數指標 ptr
	ptr = (float*)malloc(n * sizeof(float));                                // 核心：向系統申請 n 個 float 大小的動態空間
	for (int i = 0; i < n; i++) {                                           // 迴圈讀取輸入資料
		cin >> *(ptr + i);                                                  // 利用指標偏移 (Pointer Arithmetic) 存入數值
	}
	MAX(n, ptr);                                                            // 呼叫函數找出最大值
	ALL(n, ptr);                                                            // 呼叫函數計算平均值
	free(ptr);                                                              // 重要：程式結束前釋放動態申請的記憶體，避免洩漏
	return 0;                                                               // 程式順利結束
}
```
